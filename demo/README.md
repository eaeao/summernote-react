# Demo — @eaeao/summernote-react

A live Vite + React app that runs the editor straight from `../src` (via aliases — no library build
needed), so you can hack on the editor and see changes instantly.

```bash
cd demo
yarn install
yarn dev        # http://localhost:5173
```

Showcases: the 4 themes (lite / bs3 / bs4 / bs5), the 46 locales, air mode, a small custom plugin
(🕒 insert-timestamp), controlled `value`/`onChange` HTML, the imperative `ref` API, and two editors
with different themes coexisting on one page.
